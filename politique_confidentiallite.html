Politique de confidentialité

Dernière mise à jour : 01/01/2024

La protection de vos données personnelles est une priorité pour Barflix. Cette politique de confidentialité explique quelles données nous recueillons, pourquoi nous les recueillons, comment nous les utilisons, et les droits que vous avez sur vos informations.

1. Responsable du Traitement des Données

Le responsable du traitement des données personnelles collectées sur le site Barflix est :
Barflix
Adresse : 123 Rue de Paris, 75001 Paris, France
Email : contact@barflix.com

2. Données Collectées

Nous collectons uniquement les données nécessaires pour assurer nos services et vous répondre efficacement, notamment via notre formulaire de contact. Ces données peuvent inclure :

  • Nom et prénom : pour identifier nos utilisateurs.
  • Adresse e-mail : pour correspondre avec vous.
  • Numéro de téléphone : pour des échanges rapides si nécessaire.
  • Objet de la demande et Message : pour comprendre votre demande.
  • Données techniques (adresse IP, type de navigateur, etc.) : collectées automatiquement pour des raisons de sécurité et de statistiques.

3. Utilisation des Données

Les données personnelles recueillies sont utilisées pour :

  • Répondre à vos demandes et questions via le formulaire de contact.
  • Améliorer nos services et l'expérience utilisateur sur notre site.
  • Respecter nos obligations légales et de sécurité.

Nous ne partageons vos données personnelles avec aucun tiers sans votre consentement, sauf si cela est nécessaire pour répondre à une obligation légale.

4. Durée de Conservation des Données

Vos données personnelles sont conservées uniquement pendant la durée nécessaire pour remplir les objectifs pour lesquels elles ont été collectées :

  • Les données recueillies via le formulaire de contact sont conservées pendant une période de [durée précise, ex. 1 an], sauf si une obligation légale impose une conservation plus longue.

5. Vos Droits

En vertu du RGPD, vous disposez des droits suivants concernant vos données personnelles :

  • Droit d’accès : obtenir des informations sur vos données que nous traitons.
  • Droit de rectification : demander la correction de données inexactes.
  • Droit de suppression : demander la suppression de vos données, dans certaines conditions.
  • Droit de limitation du traitement : limiter l’utilisation de vos données dans certains cas.
  • Droit d’opposition : vous opposer à certains traitements de vos données.
  • Droit à la portabilité : recevoir vos données dans un format structuré et lisible, ou les transférer à un autre responsable du traitement.

Pour exercer vos droits, vous pouvez nous contacter à : contact@barflix.com. Nous répondrons à votre demande dans un délai d'un mois.

6. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, toute divulgation, altération ou destruction.

7. Modifications de la Politique de Confidentialité

Barflix se réserve le droit de modifier cette politique de confidentialité à tout moment. Nous vous informerons de toute modification via une notification sur notre site.

8. Contact

Pour toute question concernant notre politique de confidentialité, vous pouvez nous contacter à :
Email : contact@barflix.com
Adresse : 123 Rue de Paris, 75001 Paris, France
